# protobuf-util
Utility library for protobuf
This library has multiple sub modules
1. proto-generator:
   Given java Pojo file it will help in generating .proto file.
   Below are supported features.
   - Primitive Types
        - Long
        - Int
        - Double
        - Float
        - String
        - Array
   - Generic Types
        - List
        - Map
        - Set
    - Inheritance
